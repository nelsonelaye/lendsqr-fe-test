# Run Locally

Clone the project

```bash
git clone https://github.com/nelsonelaye/lendsqr-fe-test.git
```

Cd into the project directory

```bash
cd lendsqr-fe-test
```

Install dependencies

```bash
yarn install
```

Run dev server

```bash
yarn start
```

To Build Locally

```bash
yarn build
```

To install a specific dependency

```bash
yarn add **dependency name**
```
